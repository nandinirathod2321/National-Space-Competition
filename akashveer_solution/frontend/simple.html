📡 Available APIs

  • Mission Status
    GET /api/status
    Shows satellite/debris count and fuel status
  • Current States
    GET /api/states
    All tracked objects with position, velocity, fuel
  • Orbital Elements
    GET /api/orbits
    Keplerian elements for all satellites
  • Plan Maneuver
    POST /api/maneuver/plan
    Calculate delta-V for orbital transfer
  • Execute Maneuver
    POST /api/maneuver/execute
    Perform a propulsive burn
  • Ingest Telemetry
    POST /api/telemetry
    Update satellite positions and velocities
  • Simulation Tick
    POST /api/tick
    Propagate all objects forward in time
